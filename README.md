Aby zobaczyć kod źródłowy należy pobrać plik z końcówką .py należy otworzyć go w notatniku lub notepad++. Można również uruchomić program z pliku.py ale do tego należy mieć Pythona. Jeżeli zamierzasz otworzyć program bez instalacji czegokolwiek to plik .exe bedzie najlepszym rozwiązaniem.
Programy Antywirusowe mogą blokować plik lecz to fałszywy alarm.
